--- slide4 ---
Key Features
🤖  AI-Powered Generation
Claude generates columns, types, constraints, and indexes based on your table description
🗺️  Interactive ERD
Live entity-relationship diagram updates as you add tables and accept FK relationships
🔍  Smart Reference Detection
Automatically suggests foreign key links across all tables in your project
📤  Multi-Format Export
Export as SQL DDL, JSON schema, or Markdown documentation for your team
🔐  Secure API Key Mgmt
Encrypted storage of your Anthropic API keys — never shared, never logged
📜  Schema Versioning
Every regeneration creates a snapshot — roll back or compare versions anytime

Deployment Strategy
Development

Staging
AWS ECS / GCP Cloud Run — mirrors production, CI/CD validated
Production
Kubernetes (EKS/GKE) — autoscaling, managed DB, CDN-fronted
CI/CD Pipeline: GitHub Actions → Docker Build → Registry Push → ECS/K8s Deploy → Health Check
